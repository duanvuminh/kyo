| Cấu trúc | Nghĩa tiếng Việt |
|---------|------------------|
| **～どおしだ（通しだ）** | Làm gì…suốt |
| **とみえて/とみえる** | Hình như/Dường như/Có vẻ |
| **びる** | Trông giống |
| **を限りに/限りで** | Đến hết/Hết |
| **あっての** | Bởi vì có.../Có được là nhờ... |
| **ときたら** | Ấy à.../Ấy hả... |
| **だの～だの** | Nào là... nào là |
| **これまでだ** | Chỉ đến thế thôi |
| **～せめてもの** | Kể ra cũng còn may là..., ít ra là... |
| **には…なり** | Đối với ... cũng có cái tương xứng với nó |
| **とあいまって** | Cùng với, kết hợp với |
| **〜がさいご(が最後)** | Một khi đã ... thì nhất định ... |
| **～かくして** | Như vậy |
| **ためしがない** | Chưa từng, chưa bao giờ |
| **～のは...ゆえである** | Sở dĩ... là vì |
| **～やたらに** | Một cách kinh khủng |
| **～ないではおかない** | Không thể nào không ... được |
| **わ...わ** | ... ơi là ... |
| **むやみに** | Một cách bừa bãi, một cách thiếu thận trọng; một cách thiếu suy nghĩ |
| **～べく** | Để, với mục đích... |
| **であれ～であれ** | Cho dù... hay... |
| **ことなしに** | Nếu không.../Làm... mà không làm... |
| **... とされている** | Được coi như là |
| **にいわせれば** | Theo ý kiến của... |
| **～でなくてなんだろう** | Phải là, chính là |
| **～なにしろ** | Dù gì đi nữa, dẫu có thế nào |
| **～ さも** | Rất ư là, cùng cực,... |
| **… からとて** | Dù có là, thậm chí |
| **～としたことが** | Ngạc nhiên |
| **～このぶんでは** | Cứ đà này thì, với tốc độ này thì |
