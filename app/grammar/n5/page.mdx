# 📘 Ngữ pháp JLPT N5

## 1. Trợ từ cơ bản
- **は** : chủ ngữ, chủ đề
- **が** : nhấn mạnh chủ ngữ
- **を** : tân ngữ trực tiếp
- **に** : địa điểm, thời gian, hướng
- **へ** : hướng đi (tương tự に)
- **で** : địa điểm xảy ra hành động / phương tiện
- **と** : cùng với / và
- **も** : cũng
- **の** : sở hữu
- **から** : từ / vì
- **まで** : đến (thời gian, địa điểm)
- **より** : so sánh “hơn”
- **や** : liệt kê không đầy đủ “và, như...”

---

## 2. Động từ
- **ます／ません／ました／ませんでした** : chia lịch sự hiện tại / quá khứ
- **～てください** : hãy ~
- **～てもいいです** : được phép ~
- **～てはいけません** : không được ~
- **～ています** : đang ~ / trạng thái kéo dài
- **～ましょう** : cùng ~
- **～ましょうか** : cùng ~ nhé / tôi làm ~ nhé?
- **～たいです** : muốn ~
- **～に行きます** : đi để ~
- **る → ない** : thể phủ định thông thường
- **～た** : thể quá khứ thông thường
- **～たり…たりします** : nào là ~ nào là ~
- **～ことができます** : có thể ~

---

## 3. Tính từ
- **い → くない／かった／くなかった** : chia tính từ đuôi い
- **な → じゃない／でした／じゃなかった** : chia tính từ đuôi な
- **A より B のほうが…** : so sánh A hơn B
- **A がいちばん…** : A là nhất

---

## 4. Mẫu câu cơ bản
- **A は B です** : A là B
- **A は B が好きです／わかります** : A thích B / hiểu B
- **N があります／います** : có ~ (vật / người)
- **どうですか** : thế nào?
- **いくらですか** : bao nhiêu tiền?
- **どんな N** : N như thế nào?
- **どれ／どの** : cái nào?
- **ここ／そこ／あそこ／どこ** : chỗ này / chỗ kia / chỗ kia xa / chỗ nào?
- **この／その／あの／どの** : cái này / cái kia / cái xa / cái nào?
- **なに／なん** : cái gì?
- **だれ** : ai?
- **いつ** : khi nào?
- **どうして** : tại sao?
- **どう** : thế nào?
- **いっしょに～ませんか** : cùng làm ~ nhé?
- **N から N まで** : từ ~ đến ~
- **N がほしいです** : muốn có ~
- **まだ～ていません** : vẫn chưa ~
- **N のほうが～です** : N thì ~ hơn
- **N や N など** : N và N, vân vân

---

## 5. Các mẫu câu quan trọng khác
- **～からです** : vì ~
- **～けど／が** : nhưng mà ~
- **～でしょう** : có lẽ ~
- **～と思います** : tôi nghĩ rằng ~
- **～と言います** : nói rằng ~
- **～てから** : sau khi ~
- **～のが好きです／上手です／下手です** : thích / giỏi / kém việc ~
- **～まえに** : trước khi ~
- **～あとで** : sau khi ~
- **～とき** : khi ~
